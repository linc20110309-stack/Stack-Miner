# 奇妙寻宝(Stack Miner)

一款轻量级层叠三消小游戏,微信小游戏 / 抖音小游戏 / H5 通用。

> 羊了个羊 Lite + 层叠卡片 + 三消 + 寻宝主题
> 目标:30 秒学会 · 3 分钟一局 · 无限闯关 · 轻度上瘾

---

## 一、项目简介

### 核心玩法

玩家点击宝物 → 进入背包 → 3 个相同宝物自动消除 → 清空所有宝物 → 胜利

### 失败条件

背包容量固定 7 格,当 7 格全部占满且无法形成三消时立即失败。

### 胜利条件

场上所有宝物被清除即通关。

### 产品定位

- **类型**:轻度休闲三消 + 层叠消除
- **目标用户**:18~45 岁休闲/通勤/碎片时间玩家
- **单局时长**:2~5 分钟
- **平台**:首发微信小游戏,兼容抖音小游戏 / H5

## 四、核心系统

### 4.1 卡牌系统

- **15 种宝物**(水果 / 花卉 / 物品 / 乐器 / 装饰)
- 棋盘 + 背包 + 牌堆 + 弹窗预览全部统一卡牌样式:
  - **白底圆角**(圆角 14)
  - **细绿色边框**(1.5px / 被压 1px 淡绿)
  - **加厚底部立体阴影**(offsetY 8, blur 10, 黑色 0.30)
  - **彩色 emoji 图案**(🍒🍎🍓🍇🍉🌸🌻🌼🌷🚗🍰🍕🎸🎻💎)

### 4.2 层叠布局规则

- **层级**:2-4 层堆叠(layer 0 最上层)
- **可点击判定**:仅未被覆盖(覆盖面积 < 25%)的顶层卡牌可点击
- **完全遮挡**:覆盖 ≥ 90% 不渲染(数据保留)
- **透明规则**:
  - 顶层(可点击):100% 实色 + 加厚立体阴影
  - 被压下层:0.45 alpha 发灰 + 轻盈阴影,不可点击
- **动态晋升**:上层卡牌被收集后,`board._recompute` 自动刷新下层 `clickable=true`,恢复 100% 实色

### 4.3 背包系统

- **容量**:固定 7 格,不可升级
- **三消规则**:3 个相同宝物自动消除(数量 ≥ 3 立即触发)
- **失败规则**:7 格全部占满且无法形成三消 → 失败

### 4.4 关卡体系

- **前 9 关**(level 0-8):固定布局(新手引导)
- **第 10 关起**(level 9+):无限模式(动态生成)
- **解锁规则**:通关当前关 → 解锁下一关
- **最佳成绩**:记录每关最佳时间

## 五、运行方式

### 5.1 微信小游戏

1. 用微信开发者工具打开本项目根目录
2. 项目 `appid` 已在 `project.config.json` 配置
3. 点击"编译"即可在模拟器中预览

### 5.2 H5 Web 调试

1. 启动本地静态服务器(如 `python -m http.server`)
2. 浏览器访问 `index.html`
3. 注意:H5 模式下部分微信 API 不可用,核心玩法不受影响

- ✅ Canvas 原生绘制,无第三方依赖

- 帧率:目标 60 FPS,最低 30 FPS
- 粒子数:≤ 100
- 同时动画:≤ 50
- 同时音效:≤ 3
- 启动时间:≤ 3 秒
